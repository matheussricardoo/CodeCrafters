### Getting Started

To run this project locally, you'll need Rust and Cargo installed on your machine.

```bash
# Clone the repository
git clone https://github.com/matheussricardoo/CodeCrafters.git

# Navigate to project directory
cd CodeCrafters

# Run the shell
./your_program.sh

# Or build and run directly with cargo
cargo run
```
